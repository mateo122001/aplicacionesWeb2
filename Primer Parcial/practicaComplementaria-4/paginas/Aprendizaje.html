Entidad aprendizaje

Esta entidad hace referencia al tipo de aprendizaje que se encuentran disponible en una plataforma de cursos. el aprendizaje cuenta con atributos que lo caracteriza y un id que lo diferencia. En este modelo contamos con 7 atributos principales que se presentaran a continuacion

Atributos de la entidad aprendizaje

  • Id aprendizaje
  • Id idioma
  • Id instructor
  • fecha
  • Hora
  • numero de horas
  • nivel

Los atributos de esta entidad permiten conocer informacion importante para el desarrollo de la base de datos, la llave primaria es un Id que nos servira para generar relaciones entre tablas o entidades, las relaciones pueden ser de uno a muchos o de muchos a muchos etc.
